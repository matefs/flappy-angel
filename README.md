# Para rodar o codigo:
1. Baixe os arquivos
2. Descompacte os arquivos
3. Rode em um servidor web como o XAMPP 
 (Dica: Você pode rodar utilizando o comando "npx live-server", se tiver o Nodejs instalado) 
 4. Seja feliz
